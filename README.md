# FactorMiningEngine
For engineers who need to do factor-mining.

## 代码架构说明

```
┌─────────────────────────────────────────────────────────┐
│                   因子挖掘主引擎                          │
│              FactorMiningEngine                         │
├─────────────────────────────────────────────────────────┤
│  1. VocabularyGenerator    随机字母生成 + 词汇检索          │
│  2. FactorCalculator       因子计算（支持多种类型）          │
│  3. ICAnalyzer            IC/IR计算与检验                 │
│  4. LayerBacktester       分层回测与单调性检验              │
├─────────────────────────────────────────────────────────┤
│  筛选流程:                                                │
│  随机字母 → 检索词汇 → 计算因子 → IC/IR检验(IR>0.5,胜率>60%)  │
│    → 分层回测(单调性) → 多空收益/回撤(收益>5%,回撤<10%)       │
│    → 换手率检验(与投资周期匹配) → 有效因子                    │
└─────────────────────────────────────────────────────────┘

## 关键设计要点

| 模块 | 核心功能 | 筛选标准 |
| IC/IR| 信息系数分析 | IR > 0.5, IC胜率 > 60%, IC标准差 < 0.5 |
| 分层回测 | 检验因子单调性 | 斯皮尔曼秩相关显著，收益分层单调 |
| 多空组合 | 做多最高层+做空最低层 | 年化收益 > 5%, 最大回撤 < 10% |
| 换手率| 投资周期匹配 | 换手率不过高，平均持有期与设定匹配 |

如需接入真实数据或网络API，可替换 `generate_simulated_data` 和 `web_search_enrichment` 方法。
